Maven Toys Project

Introduction

The dashboard provides targeted insights about the Toys company It visualizes data in a simple way that makes it easier for the company to do its work

  • Technology: Power Bi - SQL
  • Completed: 1/8/2024
  • Target: Presenting the data in a simple way to help the company make the right decisions to develop the company and so that the company realizes its strengths and weaknesses.

Data Preparation
The data was first cleaned with Sql and the important data we needed was collected to answer the questions we wanted to know.

Detailed Insights

    The dashboard provides a multitude of insights, derived from in-depth analysis of the data:

  • Category Analysis : Revenue and profits from each of the company's product categories.
  • Products Analysis : The revenues and profits achieved by each product in the company from highest to lowest.
  • Inventory Analysis : Quantity and value of remaining stock in each category and quantity and value of remaining stock for each product.
  • Stores Analysis : Number of units sold by each store and revenue and profits earned by each store.
